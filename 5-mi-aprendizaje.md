Comparando sus conocimientos antes de hacer la práctica con sus conocimientos después de hacer la tarea, explicar los principales aprendizajes logrados para beneficio de su formación profesional.Si solucionó un problema presentado al realizar la práctica también se debe documentar.

Después de realizar esta práctica, uno de los principales aprendizajes fue el manejo de redes en Docker, especialmente el uso de redes tipo bridge creadas por el usuario. Aprendí que estas redes permiten la comunicación entre contenedores de forma más organizada y controlada, utilizando nombres de contenedor como host en lugar de direcciones IP, lo cual facilita el desarrollo de aplicaciones compuestas por múltiples servicios, como el caso de WordPress y MySQL.

Consultar: Cómo se gestionan datos confidenciales con los secretos de Docker (Docker Secrets).

Esta funcionalidad permite almacenar información sensible, como contraseñas o claves, de forma segura dentro de Docker, evitando incluirlas directamente en el código o en variables de entorno visibles.
